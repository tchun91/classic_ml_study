Market Basket Analysis (MBA)#

A model that find combinations of items that occur together frequently in transactions and define relationships bewteen the items that people would buy.

Concepts#

Assoication Rules
A model to analyze retail basket or transaction data and are intended to identify strong rules discovered in transaction data using measures of interestingness, based on the concept of strong rules.

\[ Rule = If \space an \space item \space A \space occurs, \space then \space item \space B \space also \space occurs \space with \space a \space certain \space probability. \]
\[ X => Y, where \space X < I, Y < I \space and \space X \cup Y = 0 \]


**Apriori Algorithm**
A classical algorithm in data mining, used for mining frequent itemsets and relevant association rules. It is devised to operate on a database containing a lot of transactions, for instance, items brought by customers in a store.

Item set: A collection of one or more items. $\( I = {i_1, i_2, ..., i_k} \)$

Transaction: A record of a customer’s purchase, containing one or more items. $\( T = {t_1, t_2, ..., t_3} \)$

Support
The number of transactions with both \(X\) and \(Y\) divided by the total number of transactions. $\( supp(X => Y) = \frac{|X \cup Y|}{n} \)$

def support(X_union_Y_count, total_transaction):
  return X_union_Y_count / total_transactions

Confidence
The percentage in which Y is bought with X. $\( conf(x=>y) = \frac{supp(X \cup Y)}{supp(X)} \)$

def confidence(X_count, X_union_Y_count):
  return X_union_Y_count / X_count

Lift
The ratio of the observed support to that expected if \(X\) and \(Y\) were independent. $\( lift(X=>Y) = \frac{supp(X \cup Y)}{supp(X)supp(Y)}\)$

def lift(X_count, Y_count, X_union_Y_count, total_transaction):
  support_X = X_count / total_transactions
  support_Y = Y_count / total_transactions
  support_union = X_union_Y_count / total_transactions
  return support_union/(support_X * support_Y)

Conviction
The ratio of the expected frequency that X occurs without Y X and Y were independent divided by the observed frequency of incorrect predictions. $\( conv(X => Y) = \frac{1-supp(Y)}{1-conf(X=>Y)} \)$

def conviction(Y_count, total_transactions, confidence_value):
  support_Y = Y_count / total_transactions
  return (1-support_Y) / (1-confidence_value)

Calculate Metrics

  1. \(X\) = Number of transactions containing item X.

  2. \(Y\) = Number of transactions containing item Y.

  3. \(X \cup Y\) = Number of transactions containing both X and Y.

  4. Total transactions = Total number of rows in the dataset.

def calculate_metrics(data, X, Y):
    X_count = len(data[data[X].all(axis=1)])
    Y_count = len(data[data[Y].all(axis=1)])
    X_union_Y_count = len(data[(data[X] == 1).all(axis=1) & (data[Y] == 1).all(axis=1)])
    total_transactions = len(data)
    return X_count, Y_count, X_union_Y_count, total_transactions

Set the thresholds for some metrics (support and confidence).

min_support = 0.01
min_confidence = 0.1

  • antecedent: The item or set of items that appears before the ‘then’ in an association rule.

  • consequent: The item or set of items that appears after the ‘then’ in an association rule.

  • support: How popular an itemset is, as measured by the proportion of transactions in which an itemset appears. (How frequently a particular combination of items appears together in transactions.)

  • confidence: The likelihood of item Y being purchased when item X is purchased, expressing the conditional probability of the consequent given the antecedent.

  • lift: How much more likely item Y is purchased when item X is purchased, while controlling for how popular item Y is in general.

  • Conviction: The probability that X appears without Y if they were independent with the actual frequency of incorrect predictions.

results_df = pd.DataFrame(columns=["antecedent", "consequent", "support", "confidence", "lift", "conviction"])

for X, Y in item_pairs:
    X_count, Y_count, X_union_Y_count, total_transactions = calculate_metrics(data, [X], [Y])

    support_value = support(X_union_Y_count, total_transactions)

    if support_value < min_support:
        continue

    confidence_value = confidence(X_count, X_union_Y_count)

    if confidence_value < min_confidence:
        continue

    lift_value = lift(X_count, Y_count, X_union_Y_count, total_transactions)
    conviction_value = conviction(Y_count, total_transactions, confidence_value)

    results_df = pd.concat([results_df, pd.DataFrame({
        "antecedent": [X],
        "consequent": [Y],
        "support": [support_value],
        "confidence": [confidence_value],
        "lift": [lift_value],
        "conviction": [conviction_value]
    })], ignore_index=True)
